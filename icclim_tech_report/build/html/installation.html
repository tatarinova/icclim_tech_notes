Installation¶

Dependencies¶

To use the ICCLIM you first need to install the following python libraries:
  • NumPy
  • netCDF4
  • ctypes
  • OpenClimateGIS and its dependencies if you want to use ICCLIM inside OpenClimateGIS
  • ESMPy if you want to use regridding module

Installation (Linux)¶

Note

Make sure that Python 2.7 and GCC are installed.

  1. Go to https://github.com/tatarinova/icclim.
  2. Download the current version: click to Download ZIP.
  3. Extract the file.
  4. Go to extracted directory.
  5. Run the following commands:
gcc -fPIC -g -c -Wall ./icclim/libC.c -o ./icclim/libC.o

gcc -shared -o ./icclim/libC.so ./icclim/libC.o

[sudo] python setup.py install
  1. Check if the library is installed correctly:
>>> import icclim

To get the version of installed library, do the following:

>>> icclim.__version__
3.0

Note

ICCLIM was not tested on Windows and Mac platforms...
